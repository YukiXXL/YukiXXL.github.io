HOG Packer program

A self-made program for packing HOG files


Available files

64-bit HOG Packer 32-bit HOG Packer Source Code

Usage

Simply open the program and drag n drop the folder that you want to turn into HOG file. Once done, it will ask you if you want to make another one.

"y" for Yes; "n" for No.


Info

HOG Packer was made in Code:Blocks ver. svn 12487 with MSYS2 and it's MINGW Toolchain compiler.

Mingw toolchain versions 8.1 and below has some bug with filesystem functions. 9.0.0 and above has those bugs fixed.
